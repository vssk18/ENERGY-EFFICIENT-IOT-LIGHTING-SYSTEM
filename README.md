## 1. Problem

Many public and campus lights stay at full brightness all night, even when there is no one around.  
This project explores a realistic prototype that:

- Keeps **instant brightness** when people are present  
- Avoids **visible flicker** while dimming  
- Reduces **wasted on-time** in low-traffic hours
